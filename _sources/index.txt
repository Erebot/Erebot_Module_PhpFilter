Welcome to the documentation for Erebot_Module_PhpFilter!
=========================================================

Erebot_Module_PhpFilter is a module for `Erebot`_ that transforms its input
using a `PHP stream filter`_.

For example:

..  sourcecode:: irc

    20:37:25 <+Foobar> !filter string.rot13 V ybir CUC!
    20:37:27 < Erebot> string.rot13: I love PHP!
